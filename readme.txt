В данном репозитории хранится простое Web приложение, 
построенное с использованием следующих технологий:
* Spring MVC - для построения MVC архитектуры
* JSPX - для построения шаблона страниц
* Hibernate - для работы с БД
* Maven - для управления жизненным циклом проекта

Для сборки и запуска приложения использовались:
* Eclipse Kepler (с плагинами STS, EGit, m2e-wtp)
* Tomcat Server v7.0
* HSQLDB

Приложение настроено на работу с БД, размещенной в памяти сервера. После 
перезапуска данные, записанные в БД, будут утеряны.

Если планируется использовать приложение с другой СУБД, 
то следует поменять настройки в файле /src/main/resources/jdbc.properties
